Welcome to the To-Do List API

This API allows you to manage to-do items for your to-do list application. You can perform various operations such as creating, reading, updating, and deleting to-do items. Below you'll find a list of available endpoints and instructions on how to use them.

API Endpoints

1. Get All To-Do Items

Endpoint: /todos

Method: GET

Description: Retrieve a list of all to-do items.

Response: A JSON array of to-do item objects.


Example Request:
GET http://localhost:8080/todos

Example Response:
[
    {
        "id": "123e4567-e89b-12d3-a456-426614174000",
        "title": "Sample To-Do",
        "completed": false
    }
]

2. Get a Single To-Do Item

Endpoint: /todos/:id

Method: GET

Description: Retrieve a single to-do item by its ID.

Response: A JSON object representing the to-do item.


Example Request:
GET http://localhost:8080/todos/123e4567-e89b-12d3-a456-426614174000

Example Response:
{
    "id": "123e4567-e89b-12d3-a456-426614174000",
    "title": "Sample To-Do",
    "completed": false
}

3. Add a New To-Do Item

Endpoint: /todos

Method: POST

Description: Add a new to-do item. The item ID will be automatically generated.

Request Body (JSON):


{
    "title": "New To-Do",
    "completed": false
}

Response: A JSON object representing the newly created to-do item.


Example Request:
POST http://localhost:8080/todos
Content-Type: application/json

{
    "title": "New To-Do",
    "completed": false
}

Example Response:
{
    "id": "223e4567-e89b-12d3-a456-426614174001",
    "title": "New To-Do",
    "completed": false
}

4. Update a To-Do Item

Endpoint: /todos/:id

Method: PUT

Description: Update an existing to-do item's information by its ID.

Request Body (JSON):


{
    "title": "Updated To-Do",
    "completed": true
}

Response: A JSON object representing the updated to-do item.


Example Request:
PUT http://localhost:8080/todos/223e4567-e89b-12d3-a456-426614174001
Content-Type: application/json

{
    "title": "Updated To-Do",
    "completed": true
}

Example Response:
{
    "id": "223e4567-e89b-12d3-a456-426614174001",
    "title": "Updated To-Do",
    "completed": true
}

5. Delete a To-Do Item

Endpoint: /todos/:id

Method: DELETE

Description: Delete a to-do item by its ID.

Response: A JSON object indicating success or failure.


Example Request:
DELETE http://localhost:8080/todos/223e4567-e89b-12d3-a456-426614174001

Example Response:
{
    "message": "To-Do deleted"
}
